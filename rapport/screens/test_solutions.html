Base de données de test inexistante
DB
Symfony cherche b_iconique_test — SQLSTATE[HY000] [1049] Unknown database
Création manuelle de la BDD de test dans le container MySQL
$ docker exec b_iconique_db mysql -uroot -proot -e \
  "CREATE DATABASE IF NOT EXISTS b_iconique_test CHARACTER SET utf8mb4;"
2
DATABASE_URL manquant dans .env.test
.env
Le fichier .env.test ne définit pas DATABASE_URL → Symfony utilise le host local 127.0.0.1 inaccessible depuis Docker
Ajout de DATABASE_URL pointant vers le service Docker database:3306
# .env.test
KERNEL_CLASS='App\Kernel'
APP_SECRET='$ecretf0rt3st'
DATABASE_URL="mysql://root:root@database:3306/b_iconique_test?serverVersion=8.0"

Schéma de BDD absent — migrations
Doctrine
Tables inexistantes dans b_iconique_test → toutes les requêtes ORM échouent
Exécution des migrations Doctrine sur l'environnement de test
$ docker exec b_iconique_backend php bin/console \
  doctrine:migrations:migrate --env=test --no-interaction
